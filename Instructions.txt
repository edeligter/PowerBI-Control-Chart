Instructions:
1. Use GetData to import into PowerBI.
2. Select X for Category Data. Ensure that data is either date or numeric only.
3. Select Y for Measured Data. Enusre that Y data is only numeric.
4. Select Subgroup for Subgroup Column. This is optional for multiple subgroupings.
5. Select required standard deviation and Moving Range from Statistics menu.

NB: The Subgroup column is normally a non-aggregated column that is used to define the Subgrouping of data.
